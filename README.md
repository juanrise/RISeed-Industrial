# RISeed-Industrial VST3/AU Plugin

This project is a heavily modified clone of the original [CloudSeed](https://github.com/ValdemarOrn/CloudSeed) algorithmic reverb plugin, originally created by Valdemar Erlingsson. All credit for the brilliant core algorithms and DSP backbone goes to the original creator! 

This repository represents my first "vibe-coding" project, where the entire codebase was processed, optimized, and enhanced using Antigravity, an Agentic AI IDE.

## What Did We Do?

In simple terms, we took a fantastic standard reverb and transformed it into a more aggressive, industrial-sounding effect. Here is a non-technical summary of the changes:

- **Added "Shimmer" Effects**: We built a custom pitch-shifter that automatically takes the reverberating sound and pitches it up by a full octave (+12 semitones) within the feedback loop. This creates huge, ethereal, and metallic "shimmering" tails.
- **Implemented Hard Limiters**: To keep the aggressive new sounds from blowing up your speakers or distorting uncontrollably, we added protective limits that squash the sound gently when it gets too loud.
- **Automated Parameter Controls**: We created a system that allows the plugin to read and apply knob changes (like delay sizes and feedback levels) smoothly from a background file without crashing the digital audio workstation (DAW).
- **Bulletproofed Reliability**: We eliminated various behind-the-scenes memory crashes where the plugin was trying to remember too much audio at once, ensuring it runs reliably on all operating systems without unexpectedly quitting.
- **Automated Build Pipeline**: We configured an invisible "robot" on GitHub to automatically package and build both Windows and Mac versions of the plugin every time a change is made.
